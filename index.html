LibCSL by TheMightyComp

Simple cutscene framework in XNA

LibCSL is an XNA-based framework designed to allow the implementation of in-game cutscenes in an easy and extensible way. Using CSL (Cutscene Scripting Language), cutscenes can be written and parsed by the framework. At run-time, the user simply feeds the parsed Scene object to the CSLHost, and the framework automatically manages everything else.

Dependencies

LibCSL requires the .NET framework 4.0, as well as the XNA Framework 4.0

Install

No builds of LibCSL will be available until both the parser and renderer are implemented and semi-functional. Until then, you'll just have to download the source (its written in C#) and compile it yourself.
